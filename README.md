### Get started

```bash
git clone git@github.com:Ujs74wiop6/rest-api_node_typescript.git
```

Install yarn via npm

```bash
npm install --global yarn
```

Install dependencies of the project

```bash
yarn install
```

Run the project

```bash
yarn start
```

Run test of the project

```bash
yarn test
```
